## 주요 트러블슈팅

### Git push 충돌
- **문제**: 원격에 CNAME, README.md가 이미 존재하여 push 거절
- **해결**: `git pull origin main --allow-unrelated-histories`

### WebFetch 실패
- **문제**: Databricks 블로그 페이지가 JS 렌더링이라 HTML 가져오기 실패
- **해결**: WebSearch로 검색 결과에서 콘텐츠 수집 후 직접 작성

### OG 이미지 미리보기 불가
- **문제**: 카카오 디버거에서 OG 이미지 로딩 안 됨
- **원인**: 도메인 URL 오타 (`finetuning` → `fine-tuning`)
- **해결**: 전체 프로젝트 URL 일괄 수정 + OG 이미지 재생성
