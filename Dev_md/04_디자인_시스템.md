## CSS 변수 체계

### 컬러 팔레트 (Light Mode 기본값)
```css
--bg-white: #FFFFFF;
--bg-light-gray: #F8F9FA;
--bg-medium-gray: #E9ECEF;
--text-primary: #1A1A2E;
--text-secondary: #4A4A68;
--text-light: #8A8AA3;
--border-light: #E5E7EB;
--border-medium: #D1D5DB;
```

## 반응형 브레이크포인트
| 브레이크포인트 | 대상 |
|--------------|------|
| 1200px | 데스크톱 → 모바일 네비게이션 전환 |
| 1024px | 태블릿 가로 |
| 768px | 태블릿 세로 / 대형 폰 |
| 480px | 소형 폰 |

## 다크모드
`[data-theme="dark"]` 속성으로 전환:
- 배경: #111827 → #030712
- 텍스트: #F9FAFB
- 보더: #374151
- 글래스: rgba(17, 24, 39, 0.85)

## 애니메이션
- **AOS (Animate on Scroll)**: `data-aos` 속성 기반
  - fade-up, fade-down, fade-left, fade-right, zoom-in
  - 딜레이: 100ms ~ 500ms
- **키프레임**: fadeIn, slideUp, pulse, shimmer (스켈레톤)
- **페이지 전환**: page-enter → page-enter-active
- **히어로 배경**: Particles, NetworkNodes, RisingOrbs, GeometricShapes

## 컴포넌트 스타일
- **버튼**: btn, btn-primary, btn-secondary, btn-sm, btn-lg
- **카드**: card (border + shadow)
- **폼**: form-input, form-select, form-textarea
- **뱃지**: badge, chip
- **토스트**: success, error, warning, info (border-left 컬러)
