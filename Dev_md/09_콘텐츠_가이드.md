# FineTuning - 콘텐츠 가이드

## 콘텐츠 구조

모든 레슨 콘텐츠는 `src/content/` 디렉토리에 카테고리별로 분류됩니다.

### 파일 형식
```js
// src/content/{categorySlug}/{lessonSlug}.js
export default {
  contentKo: `# 제목 (한국어)
## 섹션
내용...
`,
  contentEn: `# Title (English)
## Section
Content...
`,
};
```

- **contentKo**: 한국어 마크다운 콘텐츠
- **contentEn**: 영어 마크다운 콘텐츠
- react-markdown + remark-gfm으로 렌더링 (GFM 테이블, 코드블록 지원)

## 레슨 설정 연동

`src/config/lessons.js`에서 카테고리와 레슨 메타데이터를 관리합니다:
```js
export const CATEGORIES = [
  {
    slug: 'fundamentals',
    lessons: [
      { slug: 'what-is-finetuning', titleKo: '...', titleEn: '...' },
      // ...
    ]
  },
  // ...
];
```

- `slug`가 파일 경로와 일치해야 함: `src/content/{category.slug}/{lesson.slug}.js`
- `LessonDetail.jsx`에서 동적 import로 콘텐츠 로드

## 콘텐츠 작성 가이드

1. **마크다운 문법**: GFM (GitHub Flavored Markdown) 지원
2. **코드 블록**: ````python`, ````bash`, ````json` 등 언어 지정
3. **테이블**: GFM 테이블 문법 사용
4. **이스케이프**: 백틱은 `\\\`` 으로 이스케이프
5. **한/영 동시 작성**: contentKo와 contentEn 모두 필수
6. **다음 단계 링크**: 각 레슨 마지막에 다음 레슨 안내
