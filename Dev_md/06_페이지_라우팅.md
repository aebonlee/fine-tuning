# FineTuning - 페이지 & 라우팅

## 라우트 맵

| 경로 | 컴포넌트 | 보호 | 설명 |
|------|----------|------|------|
| `/` | Home | - | 메인 랜딩 페이지 |
| `/intro` | IntroPage | - | 사이트 소개 (About, Guide, Roadmap, Tools) |
| `/lessons` | LessonCategories | - | 전체 카테고리 목록 |
| `/lessons/:categorySlug` | LessonList | - | 카테고리별 레슨 목록 |
| `/lessons/:categorySlug/:lessonSlug` | LessonDetail | - | 레슨 상세 (마크다운) |
| `/dashboard` | Dashboard | AuthGuard | 사용자 대시보드 |
| `/settings` | Settings | AuthGuard | 프로필 설정 |
| `/login` | Login | - | 로그인 |
| `/register` | Register | - | 회원가입 |
| `/forgot-password` | ForgotPassword | - | 비밀번호 찾기 |
| `/community/board` | Board | - | 게시판 목록 |
| `/community/board/:id` | BoardDetail | - | 게시글 상세 |
| `/community/board/write` | BoardWrite | AuthGuard | 글쓰기 |
| `*` | NotFound | - | 404 페이지 |

## Lazy Loading
모든 페이지 컴포넌트는 `React.lazy()`로 동적 임포트:
```javascript
const Home = lazy(() => import('../pages/Home'));
```

## 페이지별 주요 기능

### Home.jsx
- 히어로 캐러셀 (5개 배경 애니메이션)
- 하이라이트 섹션 (3개 카드)
- 학습 경로 로드맵 (4단계)
- 커리큘럼 그룹별 카테고리 카드
- 통계 섹션 (카테고리/레슨/테마/언어)
- CTA 섹션

### LessonDetail.jsx
- 마크다운 렌더링 (ReactMarkdown + remarkGfm)
- 코드 블록 복사 기능
- TOC (목차) 자동 추출 → 사이드바 전달
- Prev/Next 네비게이션
- Breadcrumb + JSON-LD 구조화 데이터

### Board.jsx
- 카테고리 필터 (전체/공지/팁/질문/자유/쇼케이스)
- 검색 기능
- 페이지네이션
- 글쓰기 (로그인 시)

### IntroPage.jsx
- 사이드바 네비게이션 (About / Guide / Roadmap / Tools)
- About: 특징 카드 4개 + 바로가기 링크
- Guide: 6단계 사용가이드
- Roadmap: 입문→중급→고급 3단계 로드맵
- Tools: 파인튜닝 도구 소개 5개
