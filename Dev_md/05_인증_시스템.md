## 인증 방식

### 1. Google OAuth
```javascript
signInWithGoogle()
// → supabase.auth.signInWithOAuth({ provider: 'google' })
```

### 2. Kakao OAuth
```javascript
signInWithKakao()
// → supabase.auth.signInWithOAuth({ provider: 'kakao' })
```

### 3. 이메일/비밀번호
```javascript
signInWithEmail(email, password)
signUpWithEmail(email, password, metadata)
resetPassword(email)
```

## 라우트 보호

### AuthGuard
- `/dashboard`, `/settings`, `/community/board/write`
- 로그인하지 않으면 `/login`으로 리다이렉트
- `from` state로 원래 경로 저장

### AdminGuard
- `/admin/*` 경로
- 관리자 이메일: `aebonlee@gmail.com`, `admin@dreamitbiz.com`
- 비관리자는 `/`로 리다이렉트

## Supabase 테이블

### finetuning_board_posts
| 컬럼 | 타입 | 설명 |
|------|------|------|
| id | uuid | PK |
| title | text | 제목 |
| content | text | 내용 |
| category | text | notice/tip/question/free/showcase |
| user_id | uuid | 작성자 |
| author_name | text | 표시 이름 |
| views | integer | 조회수 |
| comment_count | integer | 댓글 수 |
| created_at | timestamptz | 작성일 |

### finetuning_board_comments
