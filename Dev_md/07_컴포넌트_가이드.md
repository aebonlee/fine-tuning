# FineTuning - 컴포넌트 가이드

## Context Providers

### ThemeContext
```jsx
const { mode, toggleTheme, colorTheme, setColorTheme, COLOR_OPTIONS } = useTheme();
```
- **mode**: `'light'` | `'dark'` | `'auto'`
- **colorTheme**: `'blue'` | `'red'` | `'green'` | `'purple'` | `'orange'`
- 쿠키 저장: `ft-theme`, `ft-color`
- `data-theme`과 `data-color` 속성으로 CSS 변수 전환

### LanguageContext
```jsx
const { language, toggleLanguage, t } = useLanguage();
```
- **language**: `'ko'` | `'en'`
- **t(key)**: 번역 키로 텍스트 조회 (`translations.js`)
- 쿠키 저장: `ft-lang`

### AuthContext
```jsx
const { isLoggedIn, isAdmin, user, profile, loading, signOut, updateProfile } = useAuth();
```
- Supabase `onAuthStateChange` 리스너
- 프로필 자동 로딩 (`profiles` 테이블)
- 관리자 판별: `admin.js`의 이메일 목록

### ToastContext
```jsx
const toast = useToast();
toast.success('메시지');
toast.error('에러');
toast.warning('경고');
toast.info('정보');
```
- 4가지 타입: success, error, warning, info
- 자동 닫힘 (3초)
- 슬라이드 애니메이션

## 주요 컴포넌트

### SEO
```jsx
<SEO
  title="페이지 제목"
  description="설명"
  path="/경로"
  ogImage="이미지URL"
  type="website|article"
  noindex={false}
  jsonLd={구조화데이터}
/>
```

### HeroCarousel
- 5개 슬라이드 자동 순환 (6초 간격)
- 배경 종류: particles, network, orbs, geometric
- 좌/우 화살표 + 하단 dot 네비게이션
- 스크롤 인디케이터

### Navbar
- 글래스모피즘 디자인 (`backdrop-filter: blur`)
- 스크롤 시 그림자 추가 (`.scrolled`)
- 데스크톱: 수평 메뉴 7개 + 액션 버튼
- 모바일 (≤1200px): 아코디언 메뉴 + 그룹별 카테고리
- 컬러 피커 드롭다운
- 테마 토글 (auto → light → dark)
- 언어 토글 (KO ↔ EN)
- 사용자 메뉴 (로그인 시)

### LessonsSidebar
- 현재 URL 기반 활성 그룹 자동 감지
- 카테고리 아코디언 (펼치기/접기)
- 레슨 목록 링크
- TOC (활성 레슨의 h2/h3 목차)
- 모바일: `.ck-sidebar-toggle`으로 토글

### Footer
- 3컬럼 그리드: 브랜드 / 바로가기 / 연락처
- Family Site 드롭다운 (5개 사이트)
- 사업자 정보
